## datamon diamond get

Gets diamond info

### Synopsis

Performs a direct lookup of a diamond.

Prints corresponding diamond metadata if the diamond exists,
exits with ENOENT status otherwise.

```
datamon diamond get [flags]
```

### Options

```

  -h, --help             help for get
      --repo string      The name of this repository
```
